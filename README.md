# 编程学习远征指南

《编程学习远征指南》是一份帮助编程新手打破“教程地狱”的交互式学习资源，涵盖理论、实战、历史、职业规划与工具推荐，助你从入门到进阶。

## 主要模块

- **理论教科书**：学习方法与环境搭建
- **实战工作簿**：Python、Go、PHP项目进阶
- **领域编年史**：三大语言发展与哲学
- **职业路线图**：前端、后端、运维、数据科学成长路径
- **工具资源中心**：开发、测试、部署、AI与知识管理

## 技术栈

- HTML5、Tailwind CSS、Chart.js、原生JavaScript

## 使用方法

1. 克隆仓库  
   `git clone <repository-url>`
2. 浏览 `index.html` 文件
